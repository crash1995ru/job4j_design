# 📚 Курс Junior Java Developer

## 📖 О проекте
Этот репозиторий содержит учебные материалы, упражнения и проекты, выполненные в рамках обучения по программе "Junior Java Developer".

В курсе рассматриваются фундаментальные темы: структуры данных, алгоритмы, работа с базами данных, многопоточность, сборка мусора, принципы чистой архитектуры и подготовка к техническим собеседованиям.
